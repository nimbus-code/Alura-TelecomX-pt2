# Challange 2 Alura Telecom X - Estadística, regresión lineal y machine learning
El presente repositorio aloja el Notebook **Challange_3_Telecom_X.ipynb** que tiene la finalidad de poner en práctica las habilidades desarrolladas durante la etapa del curso enfocado a Data Science. A través de la estadística se generaron dos modelos de machine learning para preveer las cancelaciones de suscripciones que tendran la empresa ficticia .Se usaron los datos normalizados y limpios del anterior Challange de Allura como base para realizar el proyecto.

## Descripción del proyecto
El desarrollo de este proyecto tiene como finalidad los siguientes objetivos:

* Preparar los datos para el modelado (tratamiento, codificación, normalización).
* Realizar análisis de correlación y selección de variables.
* Entrenar dos o más modelos de clasificación.
* Evaluar el rendimiento de los modelos con métricas.
* Interpretar los resultados, incluyendo la importancia de las variables.
* Crear una conclusión estratégica señalando los principales factores que influyen en la cancelación.



## Elementos del Notebook
El archivo fue dividido en secciones para su lectura:

| Sección |Contenido |                                                               
| - | - |
| 1. Preparación de los datos|Se carga un archivo .csv|
| 2. Correlación y selección de variables | Se genera una matriz de correlación y se generan  dos gráficos acerca de la cancelación y las variables de gasto total y meses de contrato|
| 3. Modelado predictivo |Creación de modelos estadísticos para pronosticar el comportamiento de los clientes|
| 4. Análisis de la importancia de las variables|Se obtienen las variables de mayor valor para ambos modelos y verificar las que más destacan|
| 5. Conclusiones |Consideraciones finales acerca del proyecto y el pronóstico generado |
